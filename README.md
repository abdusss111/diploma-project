# diploma-project

Проект: браузерное расширение для хром https://dapmeet.kz

Основные функции: Запись и хранение онлайн встреч из Google Meet. Доступ к ним в личном кабинете, чат с ИИ для извлечения ключевых тем из встречи.
Полезен профессионалам из IT, Sales, HR у которых большое количество созвонов
По проекту уже готовы Backend, Frontend, Chrome Extension - все опубликовано и развернуто
Идея: имплементация и построение DevOps культуры в стартапах после стадии MVP
